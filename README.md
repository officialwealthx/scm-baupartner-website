# SCM Baupartner Studio

This repository contains the new premium website system for SCM Baupartner.

The goal is to build a modern, cinematic, clean and high-performance website for a Swiss construction, surface, painting, plastering, renovation and project communication brand.

This project starts from zero. Previous repositories are only used as learning references, not as a codebase.

## Core Direction

SCM Baupartner should feel like:

- Premium Swiss Surface Studio
- Modern construction partner
- Digital project communication system
- Cinematic website experience
- Local Swiss quality company

## Main Focus

- Clean premium design
- Strong mobile experience
- Real project imagery
- Green-based brand system
- Offer request flow
- Visible customer login
- Future portal and app preview
- SEO-ready architecture
- No generic AI-template look

## Current Phase

Strategy, design system, inspiration research and master planning.
No production code should be created before the full masterplan is approved.
